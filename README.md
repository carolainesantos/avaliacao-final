# Project Manager

`Avaliação Final da matéria de backend.`

`3º semestre - ADS`

Objetivo: desenvolver um sistema de gerenciamento de projetos e tarefas

Projeto desenvolvido utilizando:

- node.js
- express
- bcrypt
- sequelize
- mysql

Comandos utilizados para instalação das dependencias:

```
 npm i express
 npm i bcrypt
 npm i sequelize
 npm i mysql2
```

→ Arquitetura: MVC
